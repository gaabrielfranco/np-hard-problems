O Makefile contém as diretivas para compilação.
É recomendável redirecionar as saída para um arquivo.

- optmizedBT -> Método exato. Mostra todos os passos na saída.

- optmizedBTBench -> Método exato. Mostra a instância, sua solução e o tempo gasto para obtê-la.

- greedyByIntersections -> Método aproximado. Mostra todos os passos na saída.

- greedyByIntersectionsBench -> Método exato. Mostra a instância, sua solução e o tempo gasto para obtê-la.

Para executar:
	"nome do metodo" < inputs/"nome da entrada" > output
